Description

This birthday countdown web app lets a user save specific birthdates and lets the user be aware of upcoming birthdays by providing a countdown of days until the upcoming birthday.


Current Functionality

The current deployment of our application includes a general layout, including features such as:

  • basic page navigation
  • styling(subject to change)
  • a visual preview of the the basic idea of the birthday countdown


Goals

We are working to make a variety of improvements including (but not limited to) the following:

  1. Improve the user interface
  2. Add in a login functionality to allow for each user to have their own personal countdowns
  3. Create various filter features such as categoring based on if the person is a family member, friend, etc
  4. An indicator that shows urgency of upcoming dates (through different color gradients, or dynamic icons


Stretch goal: Our main concept may also be extended to the functionality of a general calendar that can be shared between users. An example case would be a family of users can share the same calendar of birthday countdowns so no one forgets to prepare for upcoming birthdays.
